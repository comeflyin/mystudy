# 块级
  1. 默认占据一整行
  2. 可以设置宽高

# 行内块
  1. 可以同行显示
  2. 可以设置宽高

# 行内
  1. 可以同行显示
  2. 不可以设置宽高

# 浮动
  1. 脱离文档流
  2. 文字环绕
  3. 让块级元素同行显示
  4. 让行内元素可设置宽高
  5. 可以使用margin 但是不能使用margin: 0 auto

  # 清除浮动
  1. 给父容器设置高度
  2. 在最后一个浮动元素之后添加新的元素，在新元素上做 清除浮动
  3. 在父容器伪元素after上做 清楚浮动
  4. 在下方被影响的容器上做 清除浮动
  5. BFC
